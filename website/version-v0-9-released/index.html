Version V0.9 released

Posted by Christoph on 2 October 2010 | 7 Comments

Tags: release, V0.9

OK, I must admit, it took far too long time, but finally it's here: KeePassMobile V0.9! It has a bunch of long awaited features in it. But have a look on the release notes for yourself...

If you like the improvements of KeePassMobile V0.9 please flattr this version:

Release Notes V0.9

Here are the changes of release V0.9 in detail:

New features
  • Password generator
  • Automatic backup of changed/deleted password entries
  • Move password entries between groups (cut/paste entries)
  • Backup entries can be deleted forever
  • Add/Delete attachment to/from a password entry
  • Add/Change/Remove keyfile that is used for database encryption
  • New option to hide the backup folder
  • New option to change font size
  • Edit title of a password entry (using menu entry 'Edit Item')
  • Show tree 'path' in title
  • New translations available: Hebrew (thanks to Roee Rakovsky), Norwegian Nynorsk (thanks to Yngve Spjeld Landro), Polish (thanks to Miko≈Çaj Rozwadowski)
Improvements
  • Improved and stabilized algorithm for saving the KeePass database, prevents of corrupt database files when KeePassMobile crashes
  • Improved progress bar (while loading)
  • Polished look and feel of the 'nice' theme
  • Less resource usage for 'fast' theme
  • Improved translations: Italian (thanks to David Coen), Russian (thanks to Kirill Kozlov)
  • Improved platform identification
  • Close the application properly, if platform verification fails
Bug fixes
  • Fixed: If folder of last opened database is deleted, KeePassMobile can not load other databases any more
  • Fixed: Notes field can only edit up to 124 characters
  • Fixed: Bug in statistics calculation caused NullPointerException
  • Fixed: Bug when user wants to delete entry's expiry date
  • Fixed: Error when icon of entry is not available
