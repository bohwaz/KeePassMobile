KeePassMobile V0.9 LIGHT released

Posted by Christoph on 21 November 2010 | 1 Comments

Tags: release, V0.9, light

Until now, there has only be one KeePassMobile package for all java mobiles. Unfortunatelly, KeePassMobile does not run well on old/low-end java mobiles because it is very resource hungry. Therefore I have created a special package called KeePassMobile LIGHT that addresses this problem.

What is included in KeePassMobile LIGHT?
  • Includes all features of the KeePassMobile standard package
  • Includes only two languages (English and German)
  • Includes only the 'FAST' theme (that has no background image, transparency, animation, ...)
  • Some minor resource improvements

If you have an old/low-end java mobile and couldn't get KeePassMobile to run, try the LIGHT version now and leave a quick note in the forum, if it works for your mobile.

Go to the download page to download KeePassMobile LIGHT.
