First Public Beta Release V0.9

Posted by Christoph on 19 June 2010 | 6 Comments

Tags: release, beta, V0.9

In the past, I always asked volunteer testers to participate in a closed beta test of the upcoming KeePassMobile release. I want to change this for version V0.9. This is the first time a public beta release will be available for everybody, who wants to test the new features. So you can choose: if you want to have the stable and proven version of KeePassMobile, stick to V0.8.1. But if you want to test the latest improvements and cutting edge features then give V0.9-beta1 a try.

ATTENTION: THOUGH I INVESTED MUCH TIME IN TESTING KEEPASSMOBILE, A BETA VERSION IS NEVER AS STABLE AS THE FINAL RELEASE. THERE IS A SMALL BUT INCREASED RISK COMPARED TO A FINAL VERSION THAT KEEPASSMOBILE BETA CORRUPTS YOUR DATABASE. THEREFORE, ALWAYS CREATE A BACKUP OF YOUR DATABASE BEFORE USING THE BETA VERSION OF KEEPASSMOBILE!

There are two ways to install KeePassMobile V0.9 BETA:

  • Download the archive and install KeePassMobile manually: KeePassMobile-0.9-beta1.zip
  • OTA: Open the browser of your mobile phone, navigate to http://m2.keepassmobile.com and follow the instructions on the screen.

Please test the following new features:

  • Move password entry between groups (cut-paste)
  • Add/remove attachment to/from password entry
  • Password generator with several options
  • Keyfile can be added/changed/removed to/from master encryption key
  • New option to change font size
  • New option to show/hide backup folder
  • Old backup entries can be deleted
  • Further Enhanced UI Look&Feel
  • And many more small enhancements and bug fixes

Important: Please leave your feedback in the new beta release support forum, after testing the beta version.
