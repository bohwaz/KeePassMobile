Plans for V1.0

Posted by Christoph on 15 May 2011 | 8 Comments

Tags: release, V1.0

Hello again!

In the last six month I was totally occupied programming my program assignment for the Sun Certified Java Developer (SCJD) exam. Therefore I couldn't work on KeePassMobile and that is why you haven't heard anything from me in the last time. Now, that I have finished the project and send it to Oracle (still waiting for the results:) I want to bring KeePassMobile one step further: KeePassMobile V1.0!

This is, what I want to achieve for KeePassMobile V1.0:

  • In-Memory password protection: see one of the next blog entries for further explanations
  • some minor bug fixes to speed up and stabilize UI
  • later this year: special BlackBerry version of KeePassMobile
  • later this year: special Android version of KeePassMobile

If you know a bug or have a super important feature that you need urgently, then write a comment here or in the forums.

Regards
Christoph
