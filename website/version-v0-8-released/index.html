Version V0.8 released

Posted by Christoph on 3 December 2009 | 3 Comments

Tags: release, V0.8

I had not have as much time for KeePassMobile in the last couple of weeks, than I should. Sorry for that. But after all, I am very proud to release the version V0.8 of KeePassMobile today, that supports the long awaited create, update and save functionality.

ATTENTION: THOUGH I INVESTED MUCH TIME IN TESTING THE NEW SAVE FUNCTIONALITY THERE MIGHT BE A RISK OF DATABASE CORRUPTION. SO, KEEP A BACKUP OF YOUR DATABASE ELSEWHERE!

Release Notes V0.8

Here are the changes of release V0.8 in detail:

New features
  • Create, update and save KeePass 1.x databases:
    • Create new database
    • Create, update & delete password groups
    • Create, update & delete password entries
    • Change database settings: Password, Number of key encryption rounds
    • Encryption supports password, key file and password+key file
Improvements
  • More performance improvements
  • Integrated new version of UI library (LWUIT) (+performance improvement)
  • Saves the directory of the last opened database
  • Update last used database after closing
  • Theme improvements: 'Fast' theme should be faster, 'Nice' theme should be nicer ;)
  • New quick view mode: see user+password of a entry while navigating through the database tree
  • More accurate performance measurement
Bug fixes
  • Fixed: Search hangs under some circumstances
  • Fixed: Progess bar status sometimes invisible
