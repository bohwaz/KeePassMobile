Version V0.5 released

Posted by Christoph on 10 April 2009 | 21 Comments

Tags: release, V0.5

After two weeks of testing and improving the beta release, the version V0.5 of KeePassMobile was released today. Thanks to all testers, that helped me making KeePassMobile a stable and useful application (alphabetic order): Andreas, Bill, Björn, Daniel, Einar, Glyn, Lukas, Mani, Manilo, Marc, Philipp, Ricardo, Thomas, Tirta, Wietse. Thanks a lot!

Release Notes

Starting with this version, KeePassMobile is free, open source software that can be redistributed and/or modified under the terms of the GNU General Public License 3 (GPL3).

New features
  • Search database: a new search dialog allows to search for certain password entries
Improvements
  • Show expiry field, if set - supports localized date format
  • Hide "Meta-Inf" entries, that are used to store database internal data
  • Show progress bar during decryption and allow the user to cancel the process
  • Cycle through entry list with up/down cursor buttons
  • Select parent group, when go back (up the tree)
  • Switch menus to conform to mobile (Nokia) standard
  • New option for file browser to show all files
  • Show groups without content in grey color
Bug fixes
  • Stabilize the encryption process (V0.2 decryption did not work on some phones)
  • Show correct parent groups when navigating back (up the tree)
