Version V0.6 released

Posted by Christoph on 24 June 2009 | 15 Comments

Tags: release, V0.6

After some delays, the version V0.6 of KeePassMobile was released today. Thanks to all testers, that helped me making KeePassMobile a stable and useful application (alphabetic order): Bill, Francoise, Glyn, Michael, Ricardo, Roman, Tirta, Wietse. Thanks a lot!

Release Notes

Here are the changes of this release in detail:

New features
  • Keyfile decryption support: Databases that are encrypted using a keyfile (and password) can now be handled by KeePassMobile
  • Attachment support: Attachments that belong to a password entry can now be saved to the mobile memory
  • Multi-Language Support: available in 9 different languages. The following languages are supported with V0.6: Danish, Dutch, English, German, Italian, Russian, Spanish, Swedish and Ukrainian. Many thanks to all the translators: Carl, Evgeniy, Giulia, Peter, Ricardo, Roman, Wietse. If your language is not supported: new translators are welcome! Write a mail to contact@keepassmobile.com
  • User preferences: Preferences to set preferred language, date format, ...
  • Watchdog: The database will be closed automatically, after a certain time, the mobile has not been used. The timeout can be set in the preferences.
  • Search options: The database search can be controlled setting search options.
  • Statistics: KeePassMobile shows some statistics from the decryption process of a database. If you have any performance issues, this can help to identify the cause. Also there is a value "performance rating" that can be used to compare the decryption performance of your mobile to other mobiles. Please leave a note in the KeePassMobile "performance rating" forum.
Improvements
  • Performance: The performance of the decryption process has been increased. On some mobiles (Nokia S60 devices) the decryption of a database is now many times faster.
  • On some mobiles the UI can cause problems. It is now possible to switch to a much simpler UI theme.
  • Entries that have been expired are flagged with a red cross, so that they can be spotted immediately.
  • The progress bar showing up during database decryption is now much more precise.
  • During search, a progress bar shows the progress of the search.
  • It is now possible to navigate from anywhere within the password groups back to the top.
Bug fixes
  • Databases imported from eWallet are now handled correctly.
